# phrase-conjugator

Conjugates a phrase based on regular Spanish conjugation rules.

---

## Indicativo

### Presente

|                  | AR   | ER   | IR   |
| ---------------- | ---- | ---- | ---- |
| yo               | o    | o    | o    |
| tú               | as   | es   | es   |
| él/ella/Ud.      | a    | e    | e    |
| nosotros/as      | amos | emos | imos |
| vosotros/as      | áis  | éis  | ís   |
| ellos/ellas/Uds. | an   | en   | en   |
